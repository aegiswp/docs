## 1. Open the Site Editor

Navigate to **Appearance → Editor** to open the WordPress Site Editor. This is your central hub for designing every aspect of your site — headers, footers, templates, styles, and navigation.

## 2. Set Your Global Styles

<Steps>
  <Step title="Open Styles">
    In the Site Editor, click the **Styles** icon (half-filled circle) in the top-right toolbar.
  </Step>
  <Step title="Customize Colors">
    Aegis provides Primary, Neutral, and Semantic color palettes.
  </Step>
  <Step title="Set Typography">
    Choose from Lexend (body), Lexend Deca (headings), and JetBrains (code).
  </Step>
  <Step title="Adjust Spacing">
    Aegis uses a fluid spacing scale from XXS (8px) to XXL (96px).
  </Step>
</Steps>

## 3. Build Your Header

1. Go to **Appearance → Editor → Patterns → Template Parts**.
2. Edit the **Header** template part.
3. Add your site logo, navigation menu, and optional search toggle.
4. For a transparent header overlay, use the absolute positioning block support.

## 4. Create Pages with Patterns

Aegis ships with 130+ patterns across 20+ categories. To build a page:

1. Create a new **Page** in the WordPress editor.
2. Click the **Block Inserter** (+) and switch to the **Patterns** tab.
3. Browse categories: Hero, Features, CTA, Pricing, Testimonials, FAQ, and more.
4. Insert full-page patterns or combine individual section patterns.

<Tip>
For full-page pattern designs, select the **No Page Title** template via the editor sidebar to remove the default page title.
</Tip>

## 5. Set Up Navigation

1. In the Site Editor, edit your **Header** template part.
2. Add a **Navigation** block.
3. Create your menu structure — Aegis supports mega menus via the core submenu block.
4. For mobile, the navigation automatically converts to a responsive hamburger menu.

## 6. Configure Dark Mode

Aegis includes intelligent dark mode that adapts to user system preferences:

- Dark mode is **enabled by default** and respects `prefers-color-scheme`.
- To customize or disable, check the Aegis settings in the page editor sidebar.
- All color tokens automatically adapt between light and dark modes.

## 7. Add Custom Blocks

Aegis includes powerful custom blocks:

- **Modal** — Popup, off-canvas, bottom sheet, and fullscreen dialog modes.
- **Slider** — Responsive carousel with multiple slide types and transitions.
- **Toggle** — Collapsible content sections.

Search for these in the block inserter to get started.

## 8. Explore Block Variations

Extend core blocks with specialized functionality:

| Variation | What it does |
|-----------|-------------|
| **Icon** | Inline SVG icons via the Image block |
| **Grid** | CSS Grid layout via the Group block |
| **Marquee** | Scrolling content ticker via the Group block |
| **Counter** | Animated number counter via the Paragraph block |
| **Accordion List** | Collapsible items via the List block |
| **Curved Text** | Text along an SVG path via the Paragraph block |
| **Related Posts** | Auto-related posts via the Query block |

## 9. Export Your Design

Once your site is designed:

1. Open the Site Editor → click the **Options** menu (⋮) in the top-right.
2. Under **Tools**, click **Export**.
3. WordPress packages your customizations into a `.zip` file.
4. Upload this `.zip` to any other WordPress site to replicate your design.
