## Overview

Aegis Pro extends the Aegis theme with advanced block features, additional controls, and professional-grade functionality.

## Requirements

- WordPress 6.4+
- Aegis theme (active)
- PHP 8.1+
- Valid Aegis Pro license

## Enhanced Blocks

### Slider Pro

Extends `aegis/slider` with:

- Additional slide transition effects
- Advanced autoplay controls
- Custom animation timing
- Extended responsive breakpoint configuration
- Pro-specific data attributes for enhanced interactivity

### Modal Pro

Extends `aegis/modal` with:

- Additional trigger types
- Advanced animation options
- Conditional display rules

### Toggle

A new `aegis/toggle` block for:

- Show/hide content sections
- FAQ-style expandable content
- Conditional content display

## Enhanced Video Block

Aegis Pro significantly extends the core Video block (`core/video`) with:

### Engagement Features

- **Muted Autoplay** — Auto-captions on muted autoplay, animated play button overlay
- **Sticky Player** — Video follows user while scrolling, with close and back-to-video controls
- **Focus Mode** — Pause on tab blur, pause on scroll out, user preference toggle
- **Saved Progress** — Resume playback, remember volume/speed preferences (REST API or cookies)

### Advanced Features

- **Ambient Light** — Dynamic background glow effect matching video content
- **Thumbnail Previews** — Sprite-based timeline preview on hover
- **Touch Gestures** — Swipe volume/brightness/seek, pinch-to-zoom, double-tap seek
- **Video Playlist** — Default and Netflix-style playlist layouts with progress tracking

### Interactivity Store

- WordPress Interactivity API integration
- Server-side rendered state with client-side hydration
- Shared state management across video instances
- Chapter navigation and playlist controls

### Dynamic Sources

- **ACF** — Bind video source to ACF fields
- **Meta Box** — Bind to Meta Box plugin fields
- **User Meta** — Bind to current user's meta fields
- **Computed Metadata** — Auto-extract duration, dimensions, filesize, format

## Enhanced Map Block

Aegis Pro extends `aegis/map` with professional mapping features:

### Custom Markers & Clustering

- Custom marker icons (upload or URL) with configurable size and animation
- Marker clustering for dense marker sets (loads MarkerClusterer from CDN)

### Directions & Navigation

- Built-in directions with origin, destination, and travel mode (driving, walking, transit, cycling)
- Uses Google Maps DirectionsService and DirectionsRenderer

### Store Locator

- Search bar with geocoding and radius circle overlay
- Configurable radius, unit (km/miles), and CPT data source

### Geolocation

- Browser geolocation API with blue dot marker
- Optional geolocation button on the map

### Heatmap & Drawing

- Heatmap visualization layer with configurable radius and opacity
- Drawing tools with selectable modes (marker, polyline, polygon, circle, rectangle)

### Data Import

- KML layer support via URL
- GeoJSON import via URL

### Schema.org & Dynamic Markers

- LocalBusiness structured data output (business name, phone, address)
- Dynamic markers from CPT posts with lat/lng meta fields (max 100)
- Custom style JSON for map theming

## Pattern Control

Aegis Pro includes an admin interface to disable block patterns and templates from third-party plugins:

- **WooCommerce** — Disable store patterns and templates
- **LearnDash** — Disable course patterns
- **LifterLMS** — Disable membership patterns
- **Sensei LMS** — Disable lesson patterns
- **Fluent Forms** — Disable form patterns
- **Fluent Booking** — Disable booking patterns
- **Theme patterns** — Selectively disable Aegis theme patterns

## Video LMS & CRM Integrations

Aegis Pro extends the Video block with deep LMS and CRM integration for video-based course completion tracking.

### LearnDash Integration

- **Completion threshold** — Configurable percentage (10–100%) required to mark a lesson complete
- **Auto-complete** — Automatically mark the lesson complete when the threshold is reached
- **Force watch** — Prevent manual lesson completion until the video has been watched
- **Class:** `Aegis\Pro\Video\LearnDash`

### LifterLMS Integration

- **Completion threshold** — Same configurable percentage system
- **Auto-complete** — Calls `llms_mark_complete()` with a custom trigger
- **Force watch** — Hooks `llms_allow_lesson_completion` to block premature completion
- **Class:** `Aegis\Pro\Video\LifterLMS`

### FluentCRM Integration

- **On Play** — Trigger automation and apply tags when video starts
- **On Milestone** — Trigger automation at a configurable percentage (default 50%)
- **On Complete** — Trigger automation, apply/remove tags, add to lists on video completion
- **Class:** `Aegis\Pro\Video\FluentCRM`

## Block Registration

Aegis Pro registers its blocks via the `aegis/blocks` package:

- `Aegis\Blocks\Slider` — Slider block registration
- `Aegis\Blocks\Slide` — Slide block registration
- `Aegis\Pro\Blocks\Slider` — Pro slider extensions
- `Aegis\Pro\Blocks\Modal` — Pro modal extensions
- `Aegis\Pro\Blocks\Toggle` — Toggle block
- `Aegis\Pro\Blocks\Map` — Map block pro extensions

<Note>
The theme framework includes duplicate registration guards to prevent conflicts when both theme and Pro plugin register the same blocks.
</Note>

## Installation

<Steps>
  <Step title="Purchase license">
    Purchase an Aegis Pro license.
  </Step>
  <Step title="Download">
    Download the plugin from your account.
  </Step>
  <Step title="Upload">
    Upload to `wp-content/plugins/aegis-pro/`.
  </Step>
  <Step title="Activate">
    Activate via **Plugins → Installed Plugins**.
  </Step>
  <Step title="Enter license">
    Enter your license key under **Aegis → License**.
  </Step>
</Steps>

## Updates

Aegis Pro receives updates through the WordPress plugin update system. Ensure your license key is active to receive updates.
