## System Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| WordPress | 6.6+ | Latest |
| PHP | 7.4+ | 8.1+ |
| MySQL | 5.7+ | 8.0+ |
| MariaDB | 10.4+ | 10.6+ |

## Browser Support

Aegis supports all modern browsers:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Server Requirements

For optimal performance, your server should have:

- **Memory Limit**: 256MB minimum (512MB recommended)
- **Max Execution Time**: 60 seconds minimum
- **Upload Max Filesize**: 64MB recommended

## WordPress Configuration

Aegis works best with these WordPress settings:

```php
// wp-config.php recommendations
define('WP_MEMORY_LIMIT', '256M');
define('WP_MAX_MEMORY_LIMIT', '512M');
```
