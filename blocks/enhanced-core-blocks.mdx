## Overview

Aegis extends 37 WordPress core blocks with additional styling options, performance optimizations, and enhanced functionality.

## Text Blocks

### Paragraph (`core/paragraph`)

Extended with:
- Gradient text support
- Dynamic font sizes
- Advanced text formatting

### Heading (`core/heading`)

Extended with:
- Gradient and decoration support
- Extended heading styles
- Anchor link generation

### List (`core/list`)

Extended with:
- Custom list markers
- Checklist styling (uses `success-600` color)
- Accordion list variation

### Code (`core/code`)

Enhanced styling:
- JetBrains Mono font family
- 14px font size
- Rounded corners
- Background color

### Quote (`core/quote`)

Styled with:
- Left border accent
- Padding adjustments
- Citation styling

## Media Blocks

### Image (`core/image`)

Extended with:
- SVG icon support
- Border radius from theme
- Advanced image controls

### Cover (`core/cover`)

Enhanced with:
- Overlay and positioning options
- Zero margin defaults
- Advanced background controls

### Video (`core/video`)

Styled with:
- Border radius
- Background color for loading state
- Lazy loading support

### Audio (`core/audio`)

Styled with:
- Border radius
- Background color
- Consistent player styling

### Button (`core/button`)

Comprehensive styling:
- Primary color scheme
- Rounded corners
- Consistent padding
- Hover states

```css
/* Button defaults */
background: var(--wp--preset--color--primary-500);
color: var(--wp--preset--color--primary-25);
border-radius: var(--wp--custom--border--radius);
font-size: var(--wp--preset--font-size--14);
font-weight: var(--wp--custom--font-weight--semi-bold);
```
