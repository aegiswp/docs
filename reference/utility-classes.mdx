## Overview

Aegis generates a comprehensive set of CSS utility classes from `theme.json` design tokens. All classes use the `aegis-` prefix to avoid conflicts with other frameworks.

**Class:** `Aegis\Framework\DesignSystem\UtilityClasses`

The utility CSS is generated dynamically and output inline via the `Styleable` interface — only when utility classes are detected in the page content.

## Usage

Add utility classes via the **Additional CSS class(es)** field in the block sidebar, or via the `className` attribute in block markup:

```html
<!-- wp:group {"className":"aegis-flex aegis-gap-md aegis-items-center aegis-p-lg"} -->
<div class="wp-block-group aegis-flex aegis-gap-md aegis-items-center aegis-p-lg">
  <!-- content -->
</div>
<!-- /wp:group -->
```

<Tip>
Utility classes use theme.json design tokens, so they automatically adapt to your theme's spacing scale, font sizes, and shadow presets. They also work with dark mode since they reference CSS custom properties.
</Tip>
