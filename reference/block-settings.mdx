## Overview

Aegis extends WordPress blocks with additional settings through 22 classes in the `BlockSettings` namespace (`vendor/aegis/framework/src/BlockSettings/`). Each class hooks into `register_block_type_args` to add custom attributes and `render_block` to modify output.

## Box Shadow

Add shadow effects to blocks via the Styles panel.

### Presets

- None, XXS, XS, SM, MD, LG, XL, XXL

### CSS Output

```css
.wp-block-group {
    box-shadow: var(--wp--preset--shadow--md);
}
```

**Class:** `BlockSettings\BoxShadow`

## Animation

Scroll-triggered entrance animations using IntersectionObserver.

### Options

- **Animation type** — Fade, slide-up, slide-down, slide-left, slide-right, zoom-in, zoom-out
- **Duration** — Animation speed
- **Delay** — Stagger delay for sequential elements
- **Threshold** — Viewport intersection threshold

**Class:** `BlockSettings\Animation`

## CSS Filter

Apply CSS filters to blocks.

### Controls

- Blur, Brightness, Contrast, Grayscale, Saturate

**Class:** `BlockSettings\CssFilter`

## Transform

Apply CSS transforms to blocks.

### Controls

- Scale (X, Y), Rotate (degrees), Translate (X, Y), Skew (X, Y)

**Class:** `BlockSettings\Transform`

## Query Enhancements

Extends the core Query Loop block with advanced query parameters.

### Features

- **Meta queries** — Filter by custom field values with 14 comparison operators (=, !=, >, >=, <, <=, LIKE, NOT LIKE, IN, NOT IN, BETWEEN, NOT BETWEEN, EXISTS, NOT EXISTS)
- **Meta types** — CHAR, NUMERIC, DECIMAL, DATE, DATETIME, TIME, BINARY
- **Custom ordering** — Order by meta value, menu order, comment count, modified date, random
- **Multi-post-type** — Query across multiple post types simultaneously

**Class:** `BlockSettings\QueryEnhancements`

## Responsive

Controls block visibility and layout across breakpoints.

### Features

- **Hide on Mobile** — Hide blocks below configurable breakpoint
- **Hide on Desktop** — Hide blocks above configurable breakpoint
- **Reverse on Mobile** — Reverse column order on small screens
- **Responsive spacer** — Different spacing per breakpoint

**Class:** `BlockSettings\Responsive` (27KB — the largest BlockSettings class)

## Visibility

Conditional block visibility based on rules (free tier conditions).

### Conditions

- Lockdown, User Status, User Role, Date & Time, Screen Size, Browser & Device, URL Query String, Specific Users, Location

**Class:** `BlockSettings\Visibility`

## How It Works

Each BlockSettings class follows the same pattern:

1. **Attributes** — Adds custom attributes via `register_block_type_args` filter
2. **Editor UI** — Injects sidebar controls via `enqueue_block_editor_assets`
3. **Render** — Modifies block output via `render_block` filter
4. **Localized data** — Passes configuration to JS via `wp_localize_script`

```php
// Example: How BoxShadow adds its attribute
add_filter('register_block_type_args', function($args, $name) {
    $args['attributes']['boxShadow'] = [
        'type'    => 'string',
        'default' => '',
    ];
    return $args;
}, 10, 2);
```

<Tip>
All BlockSettings are available in the block sidebar under Styles or in dedicated panels. They apply to most block types automatically.
</Tip>
