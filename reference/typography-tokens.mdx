## Overview

Aegis provides comprehensive typography tokens for consistent text styling across your site.

## Font Family Tokens

| Token | Value |
|-------|-------|
| `--wp--preset--font-family--lexend` | `Lexend, var(--wp--preset--font-family--sans-serif)` |
| `--wp--preset--font-family--jetbrains` | `JetBrains, var(--wp--preset--font-family--monospace)` |

### Usage

```css
body {
    font-family: var(--wp--preset--font-family--lexend);
}

code {
    font-family: var(--wp--preset--font-family--jetbrains);
}
```

## Usage Examples

### CSS

```css
.hero-title {
    font-family: var(--wp--preset--font-family--lexend);
    font-size: var(--wp--preset--font-size--64);
    font-weight: var(--wp--custom--font-weight--bold);
    line-height: 1.2;
}

.body-text {
    font-family: var(--wp--preset--font-family--lexend);
    font-size: var(--wp--preset--font-size--16);
    font-weight: var(--wp--custom--font-weight--regular);
    line-height: 1.6;
}

.code-block {
    font-family: var(--wp--preset--font-family--jetbrains);
    font-size: var(--wp--preset--font-size--14);
}
```

### Block Editor

In block attributes:

```json
{
    "fontSize": "24",
    "fontFamily": "lexend"
}
```

## Responsive Behavior

Font sizes automatically scale on smaller viewports:

| Desktop | Mobile (approx) |
|---------|-----------------|
| 96px | 80px |
| 64px | 52px |
| 44px | 40px |
| 24px | 22px |
| 16px | 16px |

<Tip>
Use the fluid font sizes for responsive typography without media queries. The `clamp()` function handles scaling automatically.
</Tip>
