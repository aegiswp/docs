## Overview

Aegis defines custom CSS properties through `theme.json` that are available throughout your site for consistent styling.

## Border Properties

```css
--wp--custom--border--width: 1px;
--wp--custom--border--style: solid;
--wp--custom--border--color: var(--wp--preset--color--neutral-200);
--wp--custom--border--radius: 6px;
```

### Usage

```css
.my-element {
    border: var(--wp--custom--border--width) 
            var(--wp--custom--border--style) 
            var(--wp--custom--border--color);
    border-radius: var(--wp--custom--border--radius);
}
```

## Box Shadow Properties

```css
--wp--custom--box-shadow--inset: "";
--wp--custom--box-shadow--x: 0;
--wp--custom--box-shadow--y: 4px;
--wp--custom--box-shadow--blur: 16px;
--wp--custom--box-shadow--spread: 0;
--wp--custom--box-shadow--color: var(--wp--preset--color--custom-shadow, #6b728011);

/* Hover state */
--wp--custom--box-shadow--hover--inset: "";
--wp--custom--box-shadow--hover--y: 8px;
```

## Font Weight Properties

```css
--wp--custom--font-weight--black: 900;
--wp--custom--font-weight--extra-bold: 800;
--wp--custom--font-weight--bold: 700;
--wp--custom--font-weight--semi-bold: 600;
--wp--custom--font-weight--medium: 500;
--wp--custom--font-weight--regular: 400;
--wp--custom--font-weight--light: 300;
--wp--custom--font-weight--extra-light: 200;
--wp--custom--font-weight--thin: 100;
```

### Usage

```css
.heading {
    font-weight: var(--wp--custom--font-weight--semi-bold);
}

.body-text {
    font-weight: var(--wp--custom--font-weight--regular);
}
```

## Transition Properties

```css
--wp--custom--transition--property: all;
--wp--custom--transition--duration: 0.1s;
--wp--custom--transition--timing-function: ease-in;
```

### Usage

```css
.my-element {
    transition: var(--wp--custom--transition--property) 
                var(--wp--custom--transition--duration) 
                var(--wp--custom--transition--timing-function);
}
```

## Grid Gradient Properties

```css
--wp--custom--gradient-grid--color: var(--wp--preset--color--neutral-100);
--wp--custom--gradient-grid--size: 40px;
```

### Usage

```css
.custom-grid-bg {
    --wp--custom--gradient-grid--size: 60px;
    --wp--custom--gradient-grid--color: var(--wp--preset--color--primary-100);
    background: var(--wp--preset--gradient--grid);
}
```

## Overriding Custom Properties

Override in child theme `theme.json`:

```json
{
  "settings": {
    "custom": {
      "border": {
        "radius": "8px"
      }
    }
  }
}
```

Or in CSS:

```css
:root {
    --wp--custom--border--radius: 8px;
}
```

<Tip>
Custom properties cascade, so you can override them at any level: root, block, or element.
</Tip>
