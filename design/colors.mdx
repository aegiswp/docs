## Overview

Aegis features a comprehensive color system optimized for accessibility and visual hierarchy. All colors are defined in `theme.json` and available as CSS custom properties.

<Note>
Aegis disables WordPress core colors (`defaultPalette: false`) to provide a consistent, curated palette.
</Note>

## Usage in Blocks

Select colors in the block editor:

1. Select a block
2. Open the **Settings** sidebar
3. Expand the **Color** section
4. Choose from the Aegis palette

## Usage in CSS

Reference colors using CSS custom properties:

```css
.my-element {
  background-color: var(--wp--preset--color--primary-500);
  color: var(--wp--preset--color--neutral-0);
  border-color: var(--wp--preset--color--neutral-200);
}
```

## Accessibility

<Check>
All color combinations in Aegis meet WCAG 2.1 AA contrast requirements. The palette is also optimized for users with color vision deficiency.
</Check>
