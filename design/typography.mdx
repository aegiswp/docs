## Overview

Aegis features a comprehensive fluid typography system that scales intelligently across viewport sizes using CSS `clamp()` functions.

## Font Families

Aegis includes three variable fonts:

| Name | Slug | Usage | Weight Range | File |
|------|------|-------|-------------|------|
| **Lexend** | `lexend` | Body text | 300–900 | `assets/fonts/lexend.woff2` |
| **Lexend Deca** | `lexend-deca` | Headings | 300–900 | `assets/fonts/lexend-deca.woff2` |
| **JetBrains** | `jetbrains` | Code blocks | 100–900 | `assets/fonts/jetbrains.woff2` |

All fonts are variable fonts loaded locally (no external requests). They use `font-display: block` (Lexend, Lexend Deca) or `font-display: swap` (JetBrains) for optimal rendering.

<Note>
Fonts are loaded conditionally based on your selections in the Site Editor. If you don't use a font, it won't be loaded.
</Note>

### CSS Variables

```css
var(--wp--preset--font-family--lexend)
var(--wp--preset--font-family--lexend-deca)
var(--wp--preset--font-family--jetbrains)
```

## Usage in CSS

```css
.my-element {
  font-family: var(--wp--preset--font-family--lexend);
  font-size: var(--wp--preset--font-size--18);
  font-weight: var(--wp--custom--font-weight--medium);
  line-height: 1.6;
}

code {
  font-family: var(--wp--preset--font-family--jetbrains);
}
```

## Typography Settings

Aegis enables these typography controls in the block editor:

- **Custom font sizes** - Enter any size value
- **Line height** - Adjust line spacing
- **Drop cap** - Large first letter for paragraphs
- **Writing mode** - Vertical text support

<Tip>
Use the fluid font sizes for responsive typography that scales smoothly across all screen sizes without media queries.
</Tip>

## Variable Font Library

Aegis includes a curated collection of popular variable Google Fonts beyond the defaults. Fonts are conditionally loaded based on selections made in **Site Editor → Styles**, ensuring only used fonts are loaded.

## Advanced Text Formats

Aegis extends the block editor toolbar with additional text formatting options:

- **Clear Formatting** — Remove all inline styles
- **Underline** — Text underline decoration
- **Gradient Text** — Apply gradient fills to text
- **Dynamic Font Sizes** — Inline font size adjustments

## Customization

Typography can be customized via **Appearance → Editor → Styles → Typography**, or overridden in a child theme's `theme.json` under `settings.typography`.
