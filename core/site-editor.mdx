## Accessing the Site Editor

Navigate to **Appearance → Editor** in your WordPress admin to open the Site Editor.

<Note>
The Site Editor is your central hub for customizing templates, template parts, styles, and navigation.
</Note>

## Site Editor Sections

### Styles

Customize your site's global appearance:

- **Colors** - Background, text, links, and button colors
- **Typography** - Font families, sizes, and weights
- **Layout** - Content width and padding
- **Blocks** - Per-block style customizations

### Templates

Edit the structure of different page types:

| Template | Purpose |
|----------|---------|
| Index | Default fallback template |
| Single | Individual posts |
| Page | Static pages |
| Archive | Post listings |
| Search | Search results |
| 404 | Not found page |
| Front Page | Homepage |

### Template Parts

Reusable sections that appear across templates:

- **Header** - Site header with logo and navigation
- **Footer** - Site footer with links and copyright

### Patterns

Pre-designed block layouts you can insert into any page or template.

## Customizing Styles

<Steps>
  <Step title="Open Styles panel">
    Click the **Styles** icon (half-filled circle) in the Site Editor toolbar.
  </Step>
  <Step title="Choose what to customize">
    Select **Colors**, **Typography**, **Layout**, or **Blocks**.
  </Step>
  <Step title="Make your changes">
    Use the visual controls to adjust settings.
  </Step>
  <Step title="Save">
    Click **Save** to apply changes site-wide.
  </Step>
</Steps>

## Editing Templates

<Steps>
  <Step title="Navigate to Templates">
    In the Site Editor, click **Templates** in the left sidebar.
  </Step>
  <Step title="Select a template">
    Choose the template you want to edit (e.g., Single, Page, Archive).
  </Step>
  <Step title="Edit with blocks">
    Add, remove, or rearrange blocks to customize the template structure.
  </Step>
  <Step title="Save changes">
    Click **Save** to update the template.
  </Step>
</Steps>

## Exporting Your Customizations

You can export all your Site Editor customizations as a theme:

<Steps>
  <Step title="Open Options menu">
    Click the three-dot menu (⋮) in the Site Editor toolbar.
  </Step>
  <Step title="Select Export">
    Under **Tools**, click **Export**.
  </Step>
  <Step title="Download">
    WordPress will compile your customizations into a `.zip` file.
  </Step>
</Steps>

<Tip>
Use the export feature to migrate your customizations from a development site to production, or to create a custom child theme.
</Tip>

## Best Practices

- **Use Global Styles** for site-wide consistency
- **Create reusable patterns** for repeated layouts
- **Edit template parts** for headers/footers instead of individual templates
- **Test on multiple devices** using the viewport controls
