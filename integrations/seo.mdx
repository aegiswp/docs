## Overview

Aegis integrates with Rank Math SEO to ensure optimal search engine optimization support.

## Supported SEO Plugin

### Rank Math

- **Class:** `Integrations\RankMath`
- **Requirements:** Rank Math SEO (Free or Pro)

**Features:**

- Fallback meta description support
- Breadcrumb styling with Aegis tokens
- Schema markup compatibility with Aegis templates
- SEO analysis works correctly with block editor content
- Open Graph meta tags for Aegis patterns

## Meta Description Fallback

Aegis provides fallback meta descriptions when the Rank Math description is empty:

```php
add_filter( 'rank_math/frontend/description', function( $description ) {
    if ( empty( $description ) ) {
        $description = get_the_excerpt();
    }
    return $description;
} );
```

## Breadcrumb Styling

Rank Math breadcrumbs are styled with Aegis tokens:

```css
.rank-math-breadcrumb {
    font-size: var(--wp--preset--font-size--14);
    color: var(--wp--preset--color--neutral-500);
}

.rank-math-breadcrumb a {
    color: var(--wp--preset--color--neutral-600);
}

.rank-math-breadcrumb a:hover {
    color: var(--wp--preset--color--primary-500);
}
```

## Schema Markup

Aegis is compatible with Rank Math's schema markup:

- Article schema for posts
- Organization schema for site
- Breadcrumb schema
- FAQ schema (with FAQ patterns)

## Integration Hooks

Aegis integrates with Rank Math hooks:

```php
// Example: Modify title
add_filter( 'rank_math/frontend/title', function( $title ) {
    // Custom title modifications
    return $title;
} );
```

## Recommended Settings

For optimal Aegis compatibility:

- Enable breadcrumbs
- Use JSON-LD for schema
- Enable Open Graph

## Semantic HTML

Aegis uses proper semantic HTML throughout, which benefits SEO regardless of which SEO plugin you use:

- `<header>`, `<main>`, `<footer>`, `<nav>` landmarks
- Proper heading hierarchy in all templates and patterns
- `<article>` elements for post content
- Structured data-friendly markup

## Performance & SEO

Aegis's performance optimizations directly benefit SEO:

- **Zero default assets** — Faster page loads improve Core Web Vitals
- **Conditional loading** — Reduced page weight
- **Local fonts** — No external font requests
- **Optimized images** — WebP format, lazy loading

## Template SEO

All Aegis templates are structured for SEO:

- Single post templates include proper article markup
- Archive templates use appropriate heading hierarchy
- The 404 template provides helpful navigation
- Search templates are properly structured

## Customization

SEO integration styles can be customized via the Site Editor or by overriding in a child theme.
