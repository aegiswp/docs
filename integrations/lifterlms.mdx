## Overview

Aegis provides LifterLMS integration with theme colors, typography, and custom block patterns that replace default LifterLMS patterns.

## Features

- Theme colors and typography
- Custom block patterns
- Styled course elements
- Membership page support

## Styling

LifterLMS elements inherit Aegis design tokens:

### Course Cards

```css
.llms-loop-item {
    border-radius: var(--wp--custom--border--radius);
}

.llms-loop-item-content {
    padding: var(--wp--preset--spacing--sm);
}
```

### Buttons

```css
.llms-button-primary {
    background: var(--wp--preset--color--primary-500);
    color: var(--wp--preset--color--primary-25);
    border-radius: var(--wp--custom--border--radius);
}
```

### Progress

```css
.llms-progress .progress-bar-complete {
    background: var(--wp--preset--color--primary-500);
}
```

## Video Completion Tracking

`User` `Dev` `Pro`

Aegis Pro extends the Video block with LifterLMS-specific completion tracking. When a video block is placed on a lesson, it can automatically track viewing progress and mark the lesson complete.

### Features

- **Completion threshold** — Configurable percentage (10–100%, step 5) required to mark a lesson complete
- **Auto-complete** — Calls `llms_mark_complete()` with a custom `aegis_video_completion` trigger when the threshold is reached
- **Force watch** — Hooks `llms_allow_lesson_completion` to block premature completion until the video has been watched
- **Per-user tracking** — Progress stored in `aegis_lifterlms_video_progress` user meta

### Settings

In the block sidebar → Integrations → LifterLMS:

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| Enable | toggle | off | Activate LifterLMS tracking for this video |
| Threshold | range | 90% | Minimum watch percentage for completion |
| Auto-complete | toggle | on | Auto-mark lesson complete |
| Force watch | toggle | off | Block manual completion until threshold met |

### REST API

- `POST /aegis-pro/v1/lifterlms/complete` — Records video progress and triggers completion

**Class:** `AegisPro\Video\LifterLMS`
