## Overview

Aegis integrates with popular Learning Management System (LMS) plugins to provide consistent styling and layout support for online courses.

## Supported LMS Plugins

### LearnDash

- **Class:** `Integrations\LearnDash`
- **Requirements:** LearnDash 4.0+

**Features:**

- Focus Mode integration with theme-consistent styling
- Course and lesson templates styled with Aegis design tokens
- Quiz interfaces match theme styling
- Progress indicators use theme colors
- Responsive course grid layouts
- Dark mode support

#### Focus Mode

Aegis supports LearnDash Focus Mode with theme-consistent styling:

- Header matches theme design
- Sidebar uses theme colors
- Progress indicators styled
- Navigation follows theme patterns

#### Styling Examples

```css
.ld-course-list-item {
    border-radius: var(--wp--custom--border--radius);
    box-shadow: var(--wp--preset--shadow--sm);
}

.learndash-wrapper .ld-progress-bar {
    background: var(--wp--preset--color--neutral-100);
}

.learndash-wrapper .ld-progress-bar-percentage {
    background: var(--wp--preset--color--primary-500);
}

.learndash-wrapper .ld-button {
    background: var(--wp--preset--color--primary-500);
    color: var(--wp--preset--color--primary-25);
    border-radius: var(--wp--custom--border--radius);
}
```

### LifterLMS

- **Class:** `Integrations\LifterLMS`
- **Requirements:** LifterLMS 7.0+

**Features:**

- Course catalog styled with Aegis patterns
- Membership pages use theme templates
- Checkout forms match Aegis input styling
- Student dashboard integration
- Custom block patterns replace default LifterLMS patterns

#### Theme Support

Aegis declares LifterLMS theme support:

```php
add_theme_support( 'lifterlms' );
add_theme_support( 'lifterlms-quizzes' );
add_theme_support( 'lifterlms-sidebars' );
```

#### Styling Examples

## How Integrations Work

Each LMS integration:

1. **Detects the plugin** — Checks if the LMS plugin is active
2. **Loads conditionally** — Integration code runs only when the plugin is present
3. **Applies design tokens** — LMS elements receive Aegis colors, typography, and spacing
4. **Enhances templates** — LMS pages work within Aegis template structure

## Customization

LMS styles can be customized via:

- **Site Editor** — Modify global styles that cascade to LMS elements
- **CSS** — Target LMS-specific classes with Aegis design tokens
- **Child Theme** — Override integration behavior

## Aegis Pro — Video Completion Tracking

<Tip>
[Aegis Pro](/plugins/aegis-pro) extends LMS integrations with video-based course completion tracking on the core Video block.
</Tip>

- **LearnDash** — Configurable watch threshold, auto-complete lessons, force-watch before manual completion
- **LifterLMS** — Same threshold and auto-complete system with LifterLMS completion API
- **FluentCRM** — Trigger automations, apply/remove tags, and add contacts to lists on video play, milestone, and completion events

These features are configured per-video in the block editor sidebar under the **Integrations** panel. Each integration only appears when its target plugin is active.
