## Overview

Aegis provides a centralized admin panel under the **Aegis** menu in the WordPress dashboard. All theme features, blocks, and integrations are managed from this single location.

The admin panel is organized into tabs. Pro tabs appear only when Aegis Pro is active.

| Tab | Tier | Purpose |
|-----|------|---------|
| **Dashboard** | `Free` | Theme overview, version info, and quick links |
| **Blocks** | `Free` | Enable or disable individual Aegis blocks |
| **Modals** | `Free` | Manage modal blocks and view modal feature status |
| **Hooks** | `Free` | Manage hook patterns and view available hooks |
| **Conditionals** | `Free` `Pro` | Enable or disable conditional logic groups |
| **Integrations** | `Free` | Third-party plugin settings and feature toggles |
| **Pattern Control** | `Pro` | Disable third-party plugin patterns and templates |
| **Global Styles** | `Pro` | Custom CSS class manager |
| **Analytics** | `Pro` | Video analytics dashboard |

## Dashboard

The default landing page when clicking the Aegis menu. Displays:

- Theme version and update status
- Quick links to documentation and support
- Feature summary

## Blocks

Toggle individual Aegis blocks on or off. Disabled blocks are unregistered and will not appear in the block inserter.

Available blocks:

- **Countdown** (`aegis/countdown`) — Countdown timer
- **Modal** (`aegis/modal`) — Dialog overlays
- **Slider** (`aegis/slider`) — Responsive carousel
- **Map** (`aegis/map`) — Interactive Google Maps
- **Related Posts** (`aegis/related-posts`) — Taxonomy-based related content
- **Toggle** (`aegis/toggle`) — Accordion/toggle sections
- **Video** (`aegis/video`) — Enhanced video player

<Note>
When a block is disabled, its assets are not loaded and the block type is not registered.
</Note>

## Modals

A dedicated overview for the Modal block system:

- **Modal count** — Number of published modal blocks across the site
- **Active features** — Status of modal-related features (trigger types, animation, conditional display)
- **Add New** — Quick link to create a new post with a modal block
- **Manage All** — Link to the modal block list

## Hooks

Overview and management for [Hook Patterns](/features/hook-patterns):

- **Active count** — Number of enabled hook patterns
- **Add New** — Create a new hook pattern
- **Manage All** — Link to the hook patterns list table
- **Hook Reference** — Table of all available hooks with descriptions and enabled status

## Conditionals

`Free` `Pro`

Settings for the [Conditional Logic](/features/conditional-logic) system. Each condition group can be independently toggled:

| Group | Tier | Controls |
|-------|------|----------|
| **Visibility** | `Free` | Lockdown, Screen Size, Browser & Device |
| **User** | `Free` | User Status, User Role, Specific Users |
| **Schedule** | `Free` | Date & Time scheduling |
| **Pro Conditions** | `Pro` | Cookie, Referral, ACF, MetaBox, Post Meta, User Meta |

Disabling a group removes its section from the Conditionals meta box in the editor across all post types.

## Integrations

Settings for third-party plugin integrations. Features are conditionally displayed based on which plugins are active.

## Pattern Control

`Pro`

Admin interface to selectively disable block patterns and templates from third-party plugins. See [Aegis Pro → Pattern Control](/plugins/aegis-pro#pattern-control) for full details.

Supported plugins: WooCommerce, LearnDash, LifterLMS, Sensei LMS, Fluent Forms, Fluent Booking, and Aegis theme patterns.

## Global Styles

`Pro`

Admin page for creating and managing custom global CSS classes. See [Aegis Pro → Global Styles Manager](/plugins/aegis-pro#global-styles-manager) for full details.

## Analytics

`Pro`

Video analytics dashboard with play/pause/seek/complete event tracking, Google Analytics integration, and per-user viewing history. See [Aegis Pro → Video Analytics](/plugins/aegis-pro#video-analytics) for full details.

## Implementation

`Dev`

The admin panel is implemented in `Aegis\Admin\ConditionalLogicSettings`:

- Pages are registered as hidden submenu pages under the Aegis menu
- Tab navigation is rendered via PHP with active state detection
- Each tab has a dedicated render method
- Styles are loaded from `src/Admin/css/admin-settings.css` and `src/Admin/css/hook-patterns.css`
- Pro tabs are registered by `AegisPro\GlobalStyles\AdminPage`, `AegisPro\PatternControl`, and `AegisPro\Video\Analytics`
