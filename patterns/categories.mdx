## Overview

Aegis organizes patterns into WordPress's predefined categories. Multiple categories can be applied to a single pattern.

## Standard Categories

These are the default WordPress pattern categories used by Aegis:

| Category | Slug | Description |
|----------|------|-------------|
| Banner | `banner` | Full-width promotional banners and hero sections |
| Buttons | `buttons` | Button groups and CTA button layouts |
| Call to Action | `call-to-action` | Conversion-focused sections |
| Columns | `columns` | Multi-column content layouts |
| Contact | `contact` | Contact information and form layouts |
| Featured | `featured` | Highlighted content sections |
| Footer | `footer` | Site footer designs |
| Gallery | `gallery` | Image gallery layouts |
| Header | `header` | Site header designs |
| Media | `media` | Media-rich content sections |
| Pages | `pages` | Full-page layout designs |
| Posts | `posts` | Blog post display patterns |
| Pricing | `pricing` | Pricing tables and plans |
| Query | `query` | Post query and loop layouts |
| Team | `team` | Team member showcases |
| Testimonials | `testimonials` | Customer testimonial displays |
| Text | `text` | Text-focused content sections |

## Block Type Patterns

Some patterns are associated with specific block types and appear as suggestions when inserting those blocks:

| Block Type | Pattern Suggestion |
|------------|-------------------|
| `core/template-part/header` | Header patterns |
| `core/template-part/footer` | Footer patterns |
| `core/query` | Query loop patterns |
| `core/post-content` | Post content patterns |

## Template Type Patterns

Patterns can be suggested for specific templates:

| Template Slug | Description |
|---------------|-------------|
| `404` | 404 error page patterns |
| `home` | Homepage patterns |
| `single` | Single post patterns |
| `archive` | Archive page patterns |
| `search` | Search results patterns |

## Post Type Patterns

Full-page patterns can be restricted to specific post types:

```php
/**
 * Title: About Page
 * Slug: aegis/page-about
 * Categories: pages
 * Post Types: page
 */
```

## WooCommerce Categories

Aegis includes patterns for WooCommerce-specific templates:

| Template | Patterns |
|----------|----------|
| Product Archive | Shop grid layouts |
| Single Product | Product detail layouts |
| Cart | Cart page layouts |
| Checkout | Checkout page layouts |
| My Account | Account page layouts |

## Browsing Patterns

### By Category in the Editor

1. Open the Block Inserter (+).
2. Click the **Patterns** tab.
3. Use the category dropdown to filter.

### In the Site Editor

1. Go to **Appearance → Editor → Patterns**.
2. Categories are listed in the sidebar.
3. Click a category to see all patterns within it.

### Searching

Type a keyword in the Block Inserter search to find patterns by name or description.
