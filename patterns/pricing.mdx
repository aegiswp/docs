## Pricing Table

A comparison table for pricing tiers:

- Multiple pricing columns
- Feature lists
- Highlighted recommended plan
- Call-to-action buttons

    <!-- wp:column -->
      <!-- wp:group {"className":"pricing-card"} -->
        <!-- wp:heading {"level":3} -->Basic<!-- /wp:heading -->
        <!-- wp:paragraph {"className":"price"} -->

## Customizing Pricing Tables

### Plan Structure

Common pricing table elements:

- **Plan name**: Clear tier identification
- **Price**: Amount and billing period
- **Description**: Brief plan summary
- **Features**: Bulleted list of inclusions
- **CTA button**: Sign up or contact action
- **Badge**: "Popular", "Best Value", etc.

### Highlighting Plans

Make the recommended plan stand out:

- Different background color
- Border or shadow emphasis
- "Most Popular" badge
- Larger size or scale
- Primary button color

### Feature Comparison

Show feature availability:

```html
<!-- wp:list -->
<ul>
  <li>✓ Feature included</li>
  <li>✗ Feature not included</li>
  <li>10 GB Storage</li>
  <li>Unlimited Users</li>
</ul>
<!-- /wp:list -->
```

### Billing Toggle

For monthly/annual pricing:

```html
<!-- wp:group {"layout":{"type":"flex","justifyContent":"center"}} -->
  <!-- wp:paragraph -->Monthly<!-- /wp:paragraph -->
  <!-- Toggle switch (requires custom implementation) -->
  <!-- wp:paragraph -->Annual (Save 20%)<!-- /wp:paragraph -->
<!-- /wp:group -->
```

<Note>
Billing toggles require JavaScript for interactivity. Consider using a pricing plugin or custom implementation.
</Note>

## Best Practices

### Pricing Strategy

- **3 tiers**: Good, better, best
- **Anchor pricing**: Show value of middle tier
- **Clear differentiation**: Each tier should have distinct value

### Design

- **Visual hierarchy**: Guide users to preferred plan
- **Scannable features**: Easy to compare
- **Mobile responsive**: Stack cards on small screens
- **Consistent styling**: Same card structure across tiers

### Copy

- **Clear plan names**: Basic, Pro, Enterprise
- **Benefit-focused features**: What users get, not technical specs
- **Action-oriented CTAs**: "Start Free Trial", "Get Started"

<Tip>
Test your pricing page with real users. Watch where they look and what questions they have.
</Tip>

### Trust Elements

Add credibility:

- Money-back guarantee
- Security badges
- Customer logos
- Testimonials
- FAQ section below pricing
